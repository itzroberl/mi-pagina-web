# Open-source Bounty PR Starter Kit

## Quick bounty triage template

```markdown
**Title**: feat: [brief description]

**Summary**: 
Implements [what] for [what purpose]. Closes #[issue-number].

**Approach**:
- Step 1: [setup]
- Step 2: [implementation]  
- Step 3: [testing]

**Files Changed**:
- `file.py` - [description]
- `README.md` - [documentation]
```
